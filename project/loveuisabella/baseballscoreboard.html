9 NOV 2019

This project is a birthday present for one of my friends who's obsessed with the Red Sox. The goal is to build a scoreboard that updates according to the game's score/stats in real time and displays a countdown til the next Red Sox game if a game isn't currently happening. I also plan on adding more fun stuff, like lots of custom animations and scrolling messages.

I'm very excited for this project! It'll be a great learning experience for me. I have exactly zero experience with the components that this project requires, which just makes it more fun.

Here's a brief outline of the steps for this project:

  1. Order parts
    2.

    The prices aren't too bad. The matrix is about $20 and the Raspberry Pi Zero W is $10 (or $5 if Adafruit ever gets them back in stock). I'll probably need to spend a little extra when I get to building the casing, but it'll probably come out to $40 max.

  2. Figure out how to collect data from MLB API while I wait for parts
    3.

    I don't know actually know what an API is, so this step might be a little difficult down the line. I've been poking around Google and I found this thing, which looks useful but also only appears to have team/player statistics and not actual game scores, so I'll definitely have to do more digging.

  3. Connect Pi to WiFi
    4.

    This should be easy in theory. However, my friend is going to be using this at school, and our university has WiFi that you can't connect to as easily as you do with private WiFi networks at home.

  4. Connect Pi to LED matrix
    5.

    A little concerned for this step because it's going to require some soldering! I've soldered approximately once in my entire life and that went okay, so I hope this turns out okay too. The biggest thing is that I don't have a soldering iron at home. I might have to ask around and hope someone's got one.

  5. Make LED matrix display what I want it to
    6.

    I literally have no idea how difficult this is going to be but it'll likely be the hardest step, especially with all that fun custom stuff I plan on putting in. We'll get there when we get there.

  6. Make the casing
    7.

    I may be vastly underestimating this, but the casing should in theory just be wood, glue, paint, and some very accurate measurements.
